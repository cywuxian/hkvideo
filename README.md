# hksicVideo播放器插件

* 插件基于海康isc平台原生SDK进行的二次封装
* 插件为海康isc平台,flutter版播放器插件,支持Android和ios;
* 插件已在实际项目中使用,效果良好
* 插件提供了Dart版AK/SK验证代码
* 插件已适配最新版isc平台 v1.4+

### 不同状态下支持的功能不同,插件中的源码和demo注释写的很详细,有问题也可以加交流群讨论 856941179

* 预览状态下支持的功能:
* 预览
* 抓拍
* 录像
* 云台控制
* 码流平滑切换(仅支持Android设备)

* 回放状态下支持的功能:
* 回放
* 抓拍
* 录像
* 指定回放开始时间位置
